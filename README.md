# 🎯 결정꾼 (Decision Maker) - 심리학 기반 의사결정 보조 앱

결정을 어려워하는 사람들을 위한 심리학 기반 의사결정 도우미 웹앱입니다.

## 🎨 핵심 특징

### 1️⃣ **빠른 분석**
- 1초 이내의 빠른 피드백
- 간단한 UI로 누구나 쉽게 사용

### 2️⃣ **심리학적 근거 제시**
- 결정 분석에 심리학 개념 포함
- 사용자의 선택을 뒷받침하는 이론적 배경 제공

### 3️⃣ **재미있는 경험**
- 결정 뽑기 / 미션 / 결과 공개 같은 느낌의 UX
- 애니메이션과 상호작용적 요소로 몰입감 제고

### 4️⃣ **다양한 의사결정 유형 지원**
- 직업/경력
- 관계/인간관계
- 교육/진로
- 재정/투자
- 인생의 큰 변화

## 🚀 주요 기능

### 📝 고민 입력
사용자가 현재 고민을 자유롭게 입력합니다.

### 🧠 심리학적 분석
- 고민의 유형을 자동으로 분류
- 관련 심리학 개념 제시
- 선택지별 심리학적 장단점 분석

### ✨ 결정 제시
- 최적의 선택지 제시
- 선택의 이유와 근거 설명
- 신뢰도 표시

### 📚 심리학 개념
포함된 심리학 개념들:
- **만족 vs 최대화 이론**: 완벽한 선택보다 만족할 만한 선택의 행복도
- **손실회피**: 얻는 것보다 잃는 것을 더 크게 평가하는 경향
- **기회비용**: 한 선택의 대가로 포기하는 다른 선택들
- **확인 편향**: 자신의 의견을 확인하려는 경향 인식
- **후회 최소화**: 장기적으로 후회가 적을 선택 고려
- **예감 활용**: 직관의 가치 인식

## 💻 기술 스택

- **Frontend**: React 18, Next.js 15, TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Hooks
- **API**: Next.js API Routes
- **Animation**: CSS Animations

## 🔄 의사결정 분석 과정

1. **고민 입력** → 사용자가 현재 고민을 입력
2. **유형 분류** → 키워드 기반으로 고민의 유형 자동 분류
3. **심리학 분석** → 해당 유형의 심리학 개념 적용
4. **선택지 제시** → 심리학 기반 최적의 선택지 제시
5. **신뢰도 산정** → 입력된 정보의 구체성에 따른 신뢰도 계산
6. **결과 표시** → 애니메이션과 함께 결과 공개
